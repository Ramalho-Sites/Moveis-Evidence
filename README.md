# 🪵 Móveis Evidence — Landing Page de Portfólio

Landing page desenvolvida para a **Móveis Evidence** (DF), marcenaria especializada em móveis planejados sob medida.

O site funciona como uma vitrine digital, exibindo o portfólio de projetos finalizados de forma visual e profissional para clientes em potencial. Todo o conteúdo está em uma única página — apresentação, sobre a empresa, galeria de projetos e contato.

A galeria é organizada em categorias (Cozinha, Quarto, Banheiro, Sala, Projeto 3D e Outros), com carrossel de fotos, lightbox ao clicar nas imagens e navegação por swipe no celular.

🔗 **[Ver site ao vivo](https://ramalho-sites.github.io/Moveis-Evidence/public/index.html#inicio)**
